Javascript - event template


angular.module('myApp').animation('.my-class-name', function() {
  return {
    event: function(elem, done){
      //logic of animation
      done();
    }
  };
});

  • events are the same as for CSS (enter, leave, move, add, remove)
  • Use 3rd party animation libs inside! (JQuery, Greensock, Anima)
  • Remember to call done();!!
